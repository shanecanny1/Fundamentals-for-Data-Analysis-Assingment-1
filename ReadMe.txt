Project Title
Assignment 1 Fundamental of Data Analytics

Project / File Description
The file concerns the well-known Anscombe’s quartet dataset with a Jupyter notebook analysing the dataset.
There are four distinct tasks carried out in your Jupyter notebook.
1. Explainations of the background to the dataset – who created it, when it was created, and any speculation regarding
how it might have been created.
2. Plots of the interesting aspects of the dataset.
3. Calculations of the descriptive statistics of the variables in the dataset.
4. Explainations on why the dataset is interesting, referring to the plots and statistics within.
